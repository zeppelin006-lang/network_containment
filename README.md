# ネットワーク封じ込めゲーム MVP

ブラウザで動く、ネットワーク上の感染拡大をテーマにしたミニゲームです。  
単純な局所ルールでも、ネットワーク構造と介入位置しだいで全体の広がり方が大きく変わることを、短いプレイで体感できるようにしています。

## 概要

- ノードは人や拠点、エッジは接触や移動経路を表します
- 感染はターンごとに隣接ノードへ広がります
- プレイヤーは限られた予算で介入し、重要ノードを守ります
- 現在の固定ステージは 20 ノード構成です
- 守るべき重要ノードはリスタートごとにランダムで選ばれます
- 1プレイは約 3〜5 分を想定した軽量なMVPです

## 現在のルール

- ターン制、全 8 ターン
- 毎ターンの予算は 2
- 使用できる行動
  - `保護`：正常ノードを 2 ターン守る
  - `治療`：感染ノードを正常に戻し、1 ターン再感染を防ぐ。コストは 2
  - `接続カット`：エッジを 1 本永久に切る
- 重要ノードが 2 つ以上感染すると敗北
- 8 ターン耐えれば勝利

## 起動方法

このプロジェクトは静的ファイルだけで動きます。  
作業ディレクトリで以下を実行してください。

```bash
python3 -m http.server 8000 --directory /mnt/c/Users/t3292/Codex/web-app
```

ブラウザで以下を開きます。

```text
http://localhost:8000/index.html
```

JavaScript の構文確認だけ行う場合は次を使います。

```bash
node --check main.js
```

## 操作方法

1. 右側のアクションを選ぶ
2. ノードまたは線をクリックして介入する
3. `ターンを進める` を押して感染拡大を進行させる
4. `リスタート` で同じ初期条件から再挑戦する

## ファイル構成

- `index.html`：画面レイアウトとUIの土台
- `style.css`：見た目、レイアウト、SVG表示
- `main.js`：ゲーム状態、描画、入力、ターン進行
- `AGENTS.md`：コントリビュータ向けガイド
